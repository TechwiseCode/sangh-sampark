Generate VAPID keys (runs in your browser)

Use this when you have no SSH / Terminal on shared hosting. Keys are created on your computer — nothing is sent to any server.

  1. Click Generate keys.
  2. Copy the .env block into your server .env file (cPanel File Manager).
  3. Keep the private key secret. Delete this file from production if you upload it by mistake.
Subject: use your real email or site URL (required for push services).
